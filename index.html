Blob2Image

Convert blob URL TikTok sang file ảnh.

Không bookmark trang này bằng Ctrl+D. URL bookmark phải bắt đầu bằng javascript:.

Cài bookmarklet

Kéo nút vào bookmark bar, hoặc Copy URL rồi tạo bookmark thủ công.

Blob2Image
  1. npm run build → npm run serve
  2. Copy URL hoặc kéo nút Blob2Image vào bookmark bar
  3. Trên TikTok: click bookmark → Paste blob URL → Download
