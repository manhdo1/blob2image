Drive2PDF

Ghép trang ảnh đã render trên Google Drive thành PDF. Dán script vào Console.

Copy script

  1. npm install && npm run build:drive && npm run serve
  2. Bấm Copy script ở trên
  3. Mở PDF trên https://drive.google.com/file/d/...
  4. F12 → Console → dán → Enter
  5. Cuộn tài liệu → Quét trang → Tải PDF
